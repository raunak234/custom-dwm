Custom build of dwm - dynamic window manager


<h3>changes made:</h3> 

- added volume-control, brightness control, player control key support 

- Screenshot command                                                        

- file manager command

- gaps 

- gaps between the bar and the top thingy

- centered title

- Added emoji support by removing the iscol() function 

<h3> Patches applied </h3>

- <a href="https://dwm.suckless.org/patches/centretitle/dwm-centretitle-20200907-61bb8b2.diff">Centre Title</a>
<h3>Requirements:</h3>

- scrot (for screenshot)

- xclip (clipboard manager)

- Rofi (App menu)

- Picom-jonaburg (for the blur effect) /* optional if you dont want the blur */
